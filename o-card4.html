Design Thinking and Electronic Prototyping course

I did this course in Aalto university where we build a blackjack cheating machine. We were a team of 3. One made the CAD design and 3D printing of the machine. Another did the connection between the server and the eletronics. And I did the oython code of the algorithm to cheat the blackjack game.

As Eletronics we used steper motors, and a camera. We used Computer Vision to detect the cards and send that information for the arduino thats running the algorithm.

As extra settings we where gonna build a small device, like a watch, that would recive signals from the server to pass or continue the play of the game.
