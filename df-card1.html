Intro

Following I'm going to briefly talk about some topics and machines I learned on the Digital Fabrication minor.

For more detailed steps and explanations please visit my digital fabrication website

Computer Contolled Cutting

This week we learn how to use the Lazer Cutter machine. I decided to make a tea box. We learned the importance of the Kerf specially when making attachemnts.

Eletronics Design and Production

This week we learned how to desing our own PCB using Kicad. We also learn how to mill it using SRM machine. Finally we also learn how to solder.

CNC machine

This week we learned how to use the CNC machine. I made my design in Fusion 360 and then imported it to VCarve Pro. We learned how to set the machine and how to cut properly.

Machine Building

This week we were form into groups to build a machine together. We built the Prusa Mini 3D printer. We also had to calibrate the machine.

Output and input devices

This weeks we learned how to include an output and an input device to our PCBs. We had to make the design in Kicad, mill it and solder the components. On top of that we had to code the microcontroler (XIAO Board) using Arduino so we could use the IN/OUT devices.

Molding and Casting

This week we learned how to do molding and casting. I made a design using TinkerCad. We learned how to use the RLX-40 machine, four axis CNC machine, to CNC a piece of wax to be the master mold. We learned how to make the mixers and pour the liquides to cast.

Networking and Communications and Interface and Application Programming

This week we learned a new property of the XIAO board which is the capability of connecting with existing WiFi and Creating Access points. We had to create an interface and application programming, where I combined the PCB boards I've made with the web interface. Videos on the Digital Fabrication Website.

Wildcard week

This week we where free to choose whichever machine we wanted and i choose Glass and Ceramics. I manage to make the ceramics but I as for the glass I only made the design for the mold. I miss put it in the kiln.
